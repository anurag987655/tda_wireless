# Persistence Diagram: 

A persistence diagram is  graphical representation of topological features during the filtration. The features is represented in 2d plane as a point $(birth, death)$ where $birth$ describes when the feature first appeared and $death$ describes when the feature disappeared. The birth and death occurs through a parameter known as filtration radius $r$. The important thing in persistence diagram is the fact that $d \geq b$. So the points exists above the line $d=b$. 

# Barcode Plots: 

Barcode plot on the other hand is used to represent each features as a horizental plane line a line that stretches the feature from the point when it was born to when it disappears i.e. mathematically $[b,d)$ for the values of filtration radius $r$. They are very easy to look at and  understand than persistence diagram. On the other hand for too much features they can be messy.
